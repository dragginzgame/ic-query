# 0.4 Changelog

Detailed patch notes for the `0.4.x` release line.

Root summary: [../../CHANGELOG.md](../../CHANGELOG.md)

When a patch introduces any new CLI surface, include a fenced `bash` command
example here and in the root changelog. New CLI surface includes commands,
subcommands, options, option values, and new supported option combinations.
Cleanup-only patches without CLI behavior changes should not include command
examples.

## 0.4.1 - 2026-06-25

### Changed

- Moves the top-level process CLI runner from `ic-query` into `ic-query-cli`,
  leaving the library package focused on report APIs and family-level command
  adapters behind the `cli` feature.
- Removes now-unused library-owned top-level CLI help and command-family
  dispatch helpers after the runner move.
- Changes the `ic-query` default feature set to empty. Native host adapters and
  family-level CLI adapters now require explicit `host` or `cli` feature opt-in.

## 0.4.0 - 2026-06-25

### Added

- Adds a workspace split with `crates/ic-query` as the reusable library package
  and `crates/ic-query-cli` as the package that installs the existing `icq`
  binary.
- Adds downstream-facing public report facades for generic ICRC reports,
  NNS registry version reports, and deployed SNS list reports.
- Adds public API smoke coverage proving downstream crates can construct and
  render at least one ICRC, NNS, and SNS report without invoking CLI argument
  parsing.
- Adds `cli` and `host` features so `ic-query --no-default-features` can
  compile without CLI parsing or live host-call dependencies.

### Changed

- Moves the library package source and public API smoke tests into
  `crates/ic-query`.
- Moves the `icq` binary entrypoint and binary smoke tests into
  `crates/ic-query-cli`.
- Makes `ic-query-cli` explicitly depend on `ic-query` with the `cli` feature
  enabled.
- Updates workspace build, test, package, install, publish, and package-content
  checks for the split packages.
- Keeps low-level ICRC ledger wire helpers crate-visible while exposing the
  report-oriented ICRC request, report, live builder, and text-rendering API.

```bash
cargo install ic-query-cli
```
